# Opticon iOS

iOS companion app for [Opticon](https://github.com/nulljosh/opticon), the financial terminal.

## Features (Planned)
- Real-time market data and watchlists
- Portfolio tracking with live valuations
- Price alerts and notifications

## Stack
- SwiftUI, iOS 17+
- @Observable state management
- FMP / Yahoo Finance data (shared API with web)

## Build
Open in Xcode 15+ and run on simulator or device.
